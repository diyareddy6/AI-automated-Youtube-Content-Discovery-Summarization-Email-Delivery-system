# AI Automated YouTube Content Discovery & Summarization Email Delivery System

 About the Project
This project is an AI-powered automation system that discovers YouTube content, generates AI-based summaries, and automatically delivers the summarized content through email.

# Features
 Automated YouTube content discovery
 AI-generated content summarization
 Email delivery automation
 Workflow automation using Make platform
 API integration for AI processing
 End-to-end automated workflow

# Technologies & Tools Used
- Python
- OpenAI API
- Make Platform
- AI/ML Concepts
- Email Automation

# Project Workflow
1. Collects YouTube content automatically
2. Sends content to AI model using OpenAI API
3. Generates summarized output
4. Automates workflow using Make platform
5. Sends summarized content through email delivery

# API Integration
Integrated OpenAI API for AI-powered summarization functionality. The API was configured and used within the automated workflow for generating intelligent summaries.
